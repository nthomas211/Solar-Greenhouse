# Solar Greenhouse Simulator

A GUI for simulating the thermal performance of Chinese passive solar greenhouses. 
Calculations are performed on the backend in Python, integrating real weather data 
via API for accurate environmental parameters.

## Features

- **Simulation** — Run thermal calculations based on greenhouse geometry, materials, 
  and local weather conditions
- **History** — Simulation results are stored by date and can be viewed or exported
- **Comparison** — Compare simulation outputs against experimental data, previous 
  runs, or predictive model results

## Predictive Models (Work in Progress)

### Inverse Design Model
Takes a target indoor temperature and energy budget as input and recommends the 
greenhouse structure and material properties needed to achieve it. Trained on a 
large synthetic dataset generated by the calculation simulator, using historical 
weather data from Open-Meteo across multiple climate zones.

### Experimental Model
Trained on real sensor data collected from an on-campus Chinese greenhouse. 
Predicts the same output metrics as the calculation model but based on observed 
real-world behavior rather than physics calculations. Sensor data collection 
system currently in development.
